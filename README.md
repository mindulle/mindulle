### 🚀 Featured Projects

- **[sonagi-iac](https://github.com/mindulle/sonagi-iac)**: Infrastructure as Code (IaC) repository using Ansible to provision a K3s cluster, managing Cloudflare Tunnels, Tailscale, and automated CI/CD deployments.
- **[inkscape-mcp-server](https://github.com/mindulle/inkscape-mcp)**: Model Context Protocol (MCP) server enabling AI agents to programmatically generate and edit vector graphics using Inkscape.
- **llm-wiki (Private)**: A massive knowledge base system serving as the SSOT (Single Source of Truth) for an AI agent team, structured around a 3-Tier Architecture (Notion for CMS, Paperclip for orchestration, Metabase for BI).
- **[sonagi-design-system](https://github.com/mindulle/sonagi-design-system)**: A comprehensive design system implementing W3C Design Tokens, compiling into multi-platform assets with automated release pipelines.
- **[sonagi-anvil](https://github.com/mindulle/sonagi-anvil)**: A serverless TDD training platform and AI code review assessment environment deployed on Cloudflare Pages.
